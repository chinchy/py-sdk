# FeeLevel

Represents the fee level for a transaction, which can be set as slow, medium, or fast. Only one of fee/feeLevel is required.

## Enum

* `LOW` (value: `'LOW'`)

* `MEDIUM` (value: `'MEDIUM'`)

* `HIGH` (value: `'HIGH'`)
